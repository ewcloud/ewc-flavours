# IPA client enrollment/disenrollment automation via Morpheus

> ⚠️ These are public assets for Morpheus
Integrations/Tasks/Workflows.
If you require IPA configuration templates to customize and apply on your
own via CLI tools, checkout the 
[IPA client enroll flavour](../ipa-client-enroll-flavour/)
and [IPA client disenroll flavour](../ipa-client-disenroll-flavour/)
templates instead.

> 💡 To learn more about the automation and integration capabilities of the
Morpheus hybrid cloud management software, checkout its
[official documentation](https://docs.morpheusdata.com/en/7.0.9/integration_guides/Automation/ansible.html).

Configuration templates (i.e. [Ansible Playbooks](https://docs.ansible.com/ansible/latest/playbook_guide/playbooks.html))
deployed by the [EWC IPA enroll automation via Morpheus](https://gitlab.com/ewcloud/ewc-ansible-playbook-ipa-enroll-automation-via-morpheus)
Community Hub Item, to configure automated LDAP/DNS management of instances provisioned and
teared down via Morpheus GUI.
